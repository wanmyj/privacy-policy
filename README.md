# TinnitusAid Policy and Support Pages

This repository hosts two public pages for the iOS app:

- Privacy Policy URL: `https://<your-username>.github.io/privacy-policy/`
- Support URL: `https://<your-username>.github.io/privacy-policy/support.html`

## Files

- `index.html`: Privacy Policy page
- `support.html`: Support page for App Store Connect

## How to use in App Store Connect

Set the app metadata URLs as:

- Privacy Policy URL -> `.../privacy-policy/`
- Support URL -> `.../privacy-policy/support.html`

## Deployment (GitHub Pages)

1. Push this repository to GitHub.
2. In repository settings, enable GitHub Pages from the `main` branch (root).
3. Replace `<your-username>` above with your GitHub username (or your custom domain).
